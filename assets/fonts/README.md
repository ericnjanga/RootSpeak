# Fonts Installation

## Inter Font

Téléchargez la police Inter depuis Google Fonts :
https://fonts.google.com/specimen/Inter

Placez les fichiers suivants dans ce dossier :
- Inter-Regular.ttf
- Inter-Medium.ttf
- Inter-SemiBold.ttf
- Inter-Bold.ttf

Ou téléchargez directement depuis GitHub :
https://github.com/rsms/inter/releases

**Note :** Les fichiers .ttf doivent être placés dans ce dossier (assets/fonts/)
